En este proyecto, se integró GitHub Copilot como herramienta fundamental para la generación de la estructura base en index.html y style.css.
El desarrollo se documentó mediante un registro detallado de commits y comentarios que evidencian el flujo de trabajo asistido por IA.
La temática del sitio se centra en un emprendimiento local de plantas, cuyo objetivo es fortalecer su presencia digital y ampliar su alcance dentro de la comunidad
